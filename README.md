# What2Watch 🎬

A personalized movie recommendation platform that helps users decide **what to watch** and **where to watch**, using machine learning and real-time movie APIs.

## Features
- Movie search and details
- Streaming platform availability
- Personalized recommendations
- User watchlist and ratings
- ML-based recommendation engine

## Tech Stack
- Frontend: Next.js
- Backend: FastAPI
- Database: PostgreSQL
- ML: Python, Recommendation Systems
- API: TMDB
